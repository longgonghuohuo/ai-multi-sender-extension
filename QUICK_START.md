# 快速开始 - AI Multi Sender

## ⚡ 5 分钟上手指南

### 第一步：更新插件（1分钟）

```bash
1. 打开 Chrome 浏览器
2. 访问 chrome://extensions/
3. 找到 "AI Multi Sender"
4. 点击刷新图标 🔄
5. 等待刷新完成（无错误提示）
```

### 第二步：打开 AI 网页（2分钟）

**打开三个标签页：**

1. **ChatGPT**: https://chatgpt.com
   - 确保已登录
   - 点击"New Chat"创建新对话

2. **Claude**: https://claude.ai
   - 确保已登录
   - 进入对话页面

3. **Gemini**: https://gemini.google.com
   - 确保已登录
   - 在主页即可

### 第三步：使用插件（2分钟）

```bash
1. 点击浏览器工具栏的 AI Multi Sender 图标

2. 应该看到全页面界面，右侧显示三个 AI 的状态

3. 等待 1-2 秒，状态卡片会变成绿色

4. 在左侧输入框输入问题，例如：
   "什么是人工智能？请简单解释"

5. 点击 "🚀 发送到所有 AI" 按钮

6. 等待提示 "✅ 成功发送到 3 个页面"

7. 切换到三个 AI 标签页，检查问题是否已填充
```

## 📝 使用技巧

### 快捷键

- `Ctrl + Enter` (Windows/Linux) - 发送到所有 AI
- `Cmd + Enter` (Mac) - 发送到所有 AI
- `Ctrl + K` - 清空输入框

### 状态说明

| 颜色 | 含义 | 操作 |
|------|------|------|
| 🟢 绿色 | AI 已就绪，可以发送 | 直接使用 |
| 🔴 红色 | 未检测到，请打开网页 | 打开对应的 AI 网站 |
| ⏳ 检测中 | 正在检测（最多 500ms） | 等待完成 |

### 最佳实践

1. **保持标签页在新对话状态**
   - ChatGPT: 点击 "New Chat"
   - Claude: 创建新对话
   - Gemini: 在主页即可

2. **等待页面完全加载**
   - 看到输入框再使用插件
   - 页面加载中插件无法检测

3. **检查控制台**
   - 按 F12 打开开发者工具
   - 查看是否有 "content script loaded" 消息

## 🐛 如果遇到问题

### 快速解决方案

**问题：状态显示红色**
```bash
解决方案:
1. 刷新 AI 网页
2. 确保已登录
3. 进入对话页面
4. 等待页面加载完成
```

**问题：发送失败**
```bash
解决方案:
1. 打开 F12 控制台
2. 查看是否有 "content script loaded"
3. 如果没有，刷新页面
4. 重试发送
```

**问题：插件无响应**
```bash
解决方案:
1. 访问 chrome://extensions/
2. 刷新 AI Multi Sender 插件
3. 重启浏览器
```
