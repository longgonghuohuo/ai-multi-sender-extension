# 🔍 调试步骤

## 问题分析

你点击"发送到所有AI"按钮没反应,可能原因:

### 1. 没有打开AI标签页 ⚠️

**解决方案**: 
1. 先点击"📂 打开所有AI"按钮
2. 等待3个标签页打开(ChatGPT, Gemini, Claude)
3. 确认标签页加载完成后,再点击"发送到所有AI"

### 2. 扩展权限问题 ⚠️

**检查步骤**:
1. 打开 `chrome://extensions/`
2. 找到 AI Multi Sender
3. 确认权限中有:
   - ✅ 读取和更改您在网站上的数据
   - ✅ chatgpt.com
   - ✅ claude.ai
   - ✅ gemini.google.com

### 3. 控制台错误 🐛

**查看日志**:
1. 在扩展页面上点击右键 → 检查
2. 切换到 Console 标签
3. 点击"发送到所有AI"按钮
4. 查看是否有红色错误信息

预期应该看到:
```
[调试] sendToAll 函数被调用了！
[调试] 输入的问题: 怎么学习 ai 电商老板
[发送] 开始发送: 怎么学习 ai 电商老板
[发送] 找到 3 个AI标签页
```

## 快速测试方案

### 方案A: 使用并排窗口模式(最简单)

1. 在输入框输入问题
2. 点击"🪟 并排窗口模式"按钮
3. 等待3个窗口自动打开和填充

### 方案B: 使用标签页模式

1. **步骤1**: 点击"📂 打开所有AI"
2. **步骤2**: 等待标签页加载(约10秒)
3. **步骤3**: 输入问题
4. **步骤4**: 点击"🚀 发送到所有AI"
