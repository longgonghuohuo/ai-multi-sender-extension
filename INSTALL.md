# 安装指南

## 快速安装（5分钟完成）

### 第一步：准备文件

你现在已经有了 `ai-multi-sender-extension` 文件夹，里面包含所有必需的文件。

### 第二步：在 Chrome 中加载插件

1. **打开 Chrome 扩展页面**
   - 打开 Chrome 浏览器
   - 在地址栏输入：`chrome://extensions/`
   - 按回车键

2. **启用开发者模式**
   - 在页面右上角找到「开发者模式」开关
   - 点击开关，确保它处于「开启」状态
   - 开启后会出现三个新按钮

3. **加载插件**
   - 点击左上角的「加载已解压的扩展程序」按钮
   - 在弹出的文件选择器中，找到并选择 `ai-multi-sender-extension` 文件夹
   - 点击「选择」按钮

4. **确认安装**
   - 插件会立即加载
   - 你会在扩展列表中看到「AI Multi Sender」
   - 浏览器工具栏会出现插件图标

### 第三步：固定插件图标（推荐）

1. 点击浏览器右上角的「拼图」图标（扩展程序菜单）
2. 找到「AI Multi Sender」
3. 点击「图钉」图标，将它固定到工具栏
4. 现在你可以随时点击图标使用插件了

### 第四步：测试插件

1. **打开 AI 网站**
   - 在新标签页打开 https://chat.openai.com（或 chatgpt.com）
   - 在另一个标签页打开 https://claude.ai
   - 在第三个标签页打开 https://gemini.google.com
   - 确保你已登录这些网站

2. **使用插件**
   - 点击浏览器工具栏的 AI Multi Sender 图标
   - 在弹出窗口中输入一个问题，比如："什么是人工智能？"
   - 点击「🚀 发送到所有 AI」按钮

3. **查看结果**
   - 切换到各个 AI 标签页
   - 你会看到问题已经自动填充到输入框中
   - 点击各个网站的发送按钮即可获得回答

## 常见问题

### Q: 找不到 chrome://extensions/ 页面？

A:
- 确保使用的是 Chrome 浏览器（不是 Edge 或其他浏览器）
- 如果使用 Edge 浏览器，请访问 `edge://extensions/`
- Safari 浏览器不支持 Chrome 扩展

### Q: 没有「开发者模式」选项？

A:
- 某些企业版 Chrome 可能限制了开发者模式
- 联系你的 IT 管理员获取权限
- 或在个人电脑上尝试

### Q: 插件图标没有出现？

A:
- 点击浏览器右上角的「拼图」图标
- 在下拉菜单中找到 AI Multi Sender
- 点击图钉图标将其固定到工具栏

### Q: 插件检测不到 AI 网页？

A:
- 确保已经登录 AI 网站
- 刷新 AI 网页
- 等待页面完全加载完成
- 确保处于对话页面（不是设置页面）

### Q: 如何卸载插件？

A:
1. 访问 `chrome://extensions/`
2. 找到 AI Multi Sender
3. 点击「移除」按钮
4. 确认删除

### Q: 如何更新插件？

A:
1. 下载新版本的文件
2. 访问 `chrome://extensions/`
3. 找到 AI Multi Sender
4. 点击「刷新」图标（圆形箭头）

## 故障排查

如果遇到问题：

1. **刷新插件**
   - 进入 `chrome://extensions/`
   - 点击 AI Multi Sender 下的刷新图标

2. **查看错误信息**
   - 在 `chrome://extensions/` 页面
   - 点击 AI Multi Sender 的「详细信息」
   - 查看是否有错误提示

3. **检查控制台**
   - 在 AI 网页按 F12
   - 切换到 Console 标签
   - 查找包含 "AI Multi Sender" 的信息

4. **重新安装**
   - 移除插件
   - 重新加载插件文件夹

## 需要帮助？

如果以上方法都无法解决问题：

2. 检查浏览器版本（建议 Chrome 90+）
3. 尝试在隐身模式下测试
4. 关闭其他可能冲突的扩展
