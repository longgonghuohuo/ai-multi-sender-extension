# 🤖 AI Multi Sender

**一键向 ChatGPT、Claude、Gemini 同时发送相同问题的浏览器扩展**

![Version](https://img.shields.io/badge/version-2.0-blue)
![License](https://img.shields.io/badge/license-MIT-green)

---

## ✨ 主要功能

- ✅ **并排窗口模式** - 自动创建三个并排窗口,问题自动填充
- ✅ **标签页模式** - 发送到已打开的AI标签页
- ✅ **历史记录** - 自动保存所有提问,支持导出
- ✅ **快捷键支持** - 键盘操作更高效
- ✅ **完全本地** - 不上传任何数据,隐私安全

## 支持的 AI 平台

- ChatGPT (chat.openai.com, chatgpt.com)
- Claude (claude.ai)
- Gemini (gemini.google.com)

## 安装方法

### 方法一：开发者模式安装（推荐）

1. 打开 Chrome 浏览器
2. 在地址栏输入 `chrome://extensions/` 并回车
3. 在右上角打开「开发者模式」
4. 点击左上角的「加载已解压的扩展程序」
5. 选择 `ai-multi-sender-extension` 文件夹
6. 安装完成！插件图标会出现在浏览器工具栏

## 🚀 使用方法

### 方法一:并排窗口模式(推荐)

1. 点击浏览器工具栏的扩展图标
2. 在输入框中输入你的问题
3. 点击"🪟 并排窗口模式"按钮或按 `Ctrl+Shift+Enter`
4. 三个AI窗口会自动并排打开,问题自动填充

**优点**: 方便对比不同AI的回答

### 方法二:标签页模式

1. 点击浏览器工具栏的扩展图标
2. 点击"📂 打开所有AI"按钮
3. 等待三个AI标签页打开完成
4. 在输入框中输入你的问题
5. 点击"🚀 发送到所有AI"按钮或按 `Ctrl+Enter`

**优点**: 节省屏幕空间

## 🔧 常见问题

### Q: 发送按钮点不动?
**A**: 请先点击"打开所有AI"按钮,等待标签页加载完成

### Q: 并排窗口重叠了?
**A**: 这是正常的,窗口会自动排列在屏幕的左、中、右三个位置

### Q: 某个AI没有填充问题?
**A**: 可能是网页还没加载完成,等待几秒后手动粘贴问题即可

### Q: 支持其他AI平台吗?
**A**: 目前只支持 ChatGPT、Claude、Gemini 三个平台

## ⚠️ 注意事项

1. **首次使用**: 需要登录三个AI平台
2. **网络要求**:
   - ChatGPT 和 Claude 需要科学上网
   - Gemini 国内可直接访问
3. **权限说明**: 扩展只需要访问三个AI网站的权限
4. **隐私保护**: 所有数据本地存储,不上传到任何服务器
